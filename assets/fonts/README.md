# 中文字体支持

如果游戏中文显示出现乱码，请将中文字体文件放在这个文件夹中。

## 推荐字体文件：

1. **chinese.ttf** - 将任何支持中文的TTF字体重命名为此文件名
2. 常见的中文字体：
   - 微软雅黑 (msyh.ttf)  
   - 黑体 (simhei.ttf)
   - 宋体 (simsun.ttf)
   - 楷体 (simkai.ttf)

## 使用方法：

1. 下载一个中文TTF字体文件
2. 重命名为 `chinese.ttf`
3. 放入此文件夹
4. 重新启动游戏

## 字体来源：

可以从以下地方获取免费中文字体：
- Google Fonts (Noto Sans CJK)
- Adobe Source Han Sans  
- 系统自带字体 (Windows: C:/Windows/Fonts/)

注意：请确保使用的字体文件符合相应的使用许可。
